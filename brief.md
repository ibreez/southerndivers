This React-based web application serves as the online platform for a scuba diving center in the Southern Maldives. It features comprehensive information on diving courses (from beginner to professional levels), guided excursions to local reefs and marine sites, vacation packages, a photo gallery showcasing underwater experiences, customer reviews, team profiles, and additional services. The app includes an admin panel for managing content such as courses, excursions, packages, gallery images, testimonials, and team details, all built with modern web technologies like Vite, Tailwind CSS, and Framer Motion for a responsive and engaging user experience.
